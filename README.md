# D Moon Giveaway System

Automatically detects "Gift" / "gift" / "GIFT" comments on Facebook & Instagram posts and logs them in a dashboard.

## Environment Variables (set in Railway)

| Variable | Value |
|----------|-------|
| `VERIFY_TOKEN` | `dmoon_giveaway_2026` |
| `FB_PAGE_ACCESS_TOKEN` | (your copied FB token) |
| `IG_ACCESS_TOKEN` | (your copied IG token) |
| `IG_BUSINESS_ACCOUNT_ID` | `17841411697514213` |
| `FB_PAGE_ID` | `297209906812982` |

## Webhook URL (after deploy)
`https://YOUR-RAILWAY-URL/webhook`
